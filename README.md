# YearProgress

올해·이번 달·이번 주·오늘, 그리고 **인생**이 얼마나 지났는지 보여주는 macOS 메뉴 막대 + 위젯 앱.

- 🧭 **메뉴 막대**: 선택한 기간의 진행률(%)을 날짜·시간과 함께 표시
- 🟦 **위젯**(데스크탑/알림 센터): 점 그리드 / 원형 그래프, 월별 3×4 미니 달력(탭하면 해당 월 확대)
- 🎨 색상 커스텀(컬러 휠 + 프리셋), 다크/라이트 대응
- ⏳ 기간: 올해 / 이번 달 / 이번 주 / 오늘 / **인생**(생년월일·기대수명 설정)
- 🌐 시스템 언어(한국어/영어) 자동, 24시간제
- 🔁 로그인 시 자동 실행

## 설치 (Homebrew)

```bash
brew install --cask ChoMnSeong/tap/yearprogress
```

> 이 앱은 Apple 공증(notarization)을 받지 않은 개인 빌드입니다. cask가 설치 시 격리 속성을 자동으로 제거합니다.

## 빌드

Xcode 16+ / macOS 14+ 에서 `YearProgress.xcodeproj` 를 열어 `YearProgress` 스킴으로 실행하세요.
